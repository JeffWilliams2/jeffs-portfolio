## Overview

Built a fast, modern portfolio website to showcase data engineering projects and technical writing. Focused on performance, SEO, and clean design.

## Tech Stack

- **Astro.js** — Static site generator with partial hydration for minimal JavaScript
- **React** — Interactive components (navbar, search, theme toggle)
- **Tailwind CSS** — Utility-first styling with dark mode support
- **TypeScript** — Type safety throughout
- **Vercel** — Deployment with Edge Network CDN

## Features

- **Dark/Light mode** with system preference detection
- **Blog** with Markdown/MDX support and syntax highlighting
- **Project showcase** with detailed case studies
- **Search functionality** for blog posts
- **RSS feed** for content syndication
- **SEO optimized** with structured data, meta tags, sitemap
- **Google Analytics** integration for visitor insights
- **Vercel Speed Insights** for real-time performance monitoring

## Key Optimizations

**Fonts**: Self-hosted with `font-display: swap` and preloading critical fonts

**Images**: Lazy loading, proper sizing, next-gen formats

**JavaScript**: Minimal client-side JS through Astro's island architecture

**CSS**: Tailwind purging unused styles, critical CSS inlined
